Este é um parágrafo de texto. Aqui você pode usar as tags
<mark > é usada para destacar um trecho de texto em uma página da web. Ela é usada para indicar informações importantes ou para dar um estilo diferente ao texto.


A tag <span > no HTML é usada para definir texto menor, reduzindo o tamanho da fonte. Ela diminui o tamanho da fonte em um nível (de médio para pequeno, de x-grande para grande). Essa tag tem uma propriedade de exibição inline.
A tag <i > é usada apenas para estilizar o texto em itálico, sem dar ênfase semântica.
< u >Ela pode ser usada para envolver uma parte do texto que você deseja exibir com uma linha de sublinhado.
<strong > Ela é tipicamente usada para envolver uma parte do texto que deve ser exibida em negrito.
A tag <sub > é usada para texto subscrito (abaixo da linha). H20
A tag <sup > é usada para texto sobrescrito (acima da linha).
Você pode usar a tag <blockquote > para indicar que um bloco de texto é uma citação de outro autor, geralmente com recuo e estilo diferenciado para destacá-lo do restante do conteúdo. Aqui está um exemplo de como você pode usar a tag
